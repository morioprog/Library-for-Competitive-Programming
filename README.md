## ライブラリ追加時

### 注意点
rootの深さを0としたとき，最大の深さは3以下である必要がある．
* 深さ3のファイルの例: [CumulativeSum1D.cpp(1次元累積和)](DataStructure/CumulativeSum/CumulativeSum1D.cpp)
